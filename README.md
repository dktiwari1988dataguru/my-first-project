# My First Project

3 small Python projects collection!

## Projects

### 1. Calculator (`calculator.py`)
Simple calculator - add, subtract, multiply, divide
```
python calculator.py
```

### 2. Number Guessing Game (`number_guess.py`)
1 se 100 ke beech number guess karo - 7 chances milenge!
```
python number_guess.py
```

### 3. To-Do List (`todo_list.py`)
Tasks add karo, complete karo, delete karo - data save hota hai!
```
python todo_list.py
```

## How to Run
1. Python install hona chahiye
2. Terminal/PowerShell mein project folder mein jao
3. Upar diye gaye commands run karo
